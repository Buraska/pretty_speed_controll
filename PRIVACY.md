# Privacy Policy for Video Speed Toggle

**Last updated:** March 2025

## 1. Overview

Video Speed Toggle ("the extension") is a browser extension that lets you change video playback speed and switch between two speeds with a long-press on videos on web pages. This policy describes what data the extension uses and does not use.

## 2. Data collection

The extension **does not collect, store, or transmit any personal data or usage data**.

- Personally Identifiable Information: We do not collect any personally identifiable information.
- Health Information: We do not collect any health information.
- Financial and Payment Information: We do not collect any financial information.
- Authentication Information: We do not collect any authentication information.
- Personal Communications: We do not collect any personal communications.
- Location: We do not collect any location information.
- Web History: We do not collect any web history information.
- User Activity: We do not collect any user activity data.
- Website Content: We do not collect any content from the websites you visit.

## 3. What the extension does on your device

- **Content scripts** run only on web pages you visit (http, https, and local file pages) to find `<video>` elements and add the long-press and speed-control behavior.
- **Playback speed** is changed only in the browser (via the video element's `playbackRate`). The extension does not upload or sync speed settings.
- **Extension assets** (e.g. CSS, icons) are loaded only from the installed extension package; no code or resources are loaded from the internet.

## 4. Permissions

The extension does not request host permissions or broad access to your data. It can run on http, https, and file URLs only so it can work on pages that contain videos. It does not read your browsing history, cookies, or other personal information beyond what is needed to attach to video elements on the current page.

## 5. Third parties

The extension does not integrate third-party analytics, ads, or other services that would receive your data.

## 6. Changes to this policy

If we change this privacy policy, we will update the "Last updated" date. For an installed extension, significant changes will be reflected in the extension's listing or release notes.
